Overview

Composable GUI library on top of wxErlang.

Version: 0.1

Authors: Dmitry Golubovsky (golubovsky@gmail.com).

References

  • Fudgets

The wxErlang package is now official part of the Erlang/OTP distribution. The library itself exposes a low level imperative style interface. The wxi library provides functional-style composable GUI interface layer on top of wxErlang. The library is based on ideas found in the Haskell GUI library Fudgets.
