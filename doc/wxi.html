catchEvents/2

catchEvents(Es, Id) -> any()

Connect the parent to the list of events explicitly setting parent window Id. This may be useful when setting events capture for a panel which was created without window Id specified.

comp/2

comp(Sub, C) -> any()

Compose a subordinate piece of GUI. List of widgets encodes parallel composition, tuple of widgets encodes sequential composition. This function is intended to be called from within the creation phase code of a widget having subordinates (e. g. panel).

grid/2

grid(Cols, Sub) -> any()

Create a panel with GridSizer. Number of columns is specified upon creation; rows are added automatically as children are added. This widget does not allow reverse placement of its subordinates.

map/1

map(F) -> any()

Apply the given function to any event received, and send the result to the event link.

mapState/2

mapState(F, S) -> any()

Apply the given function fo any event received and the encapsulated state. The widget keeps the state between messages. The value returned from the function will be sent to the event link of this widget.

maybe/1

maybe(F) -> any()

Apply the given function to any event received, but pass the event to the event if the function returns a tuple {'just', E} where E is the value to be passed.

modSizerFlags/2

modSizerFlags(F, Sub) -> any()

Modify sizer flags in the context. All subordinate widgets will use the modified context. This widget only allows normal order of subordinates placement although orientation is inherited from the parent.

never/0

never() -> any()

Never pass any event.

panel/1

panel(Sub) -> any()

Create a panel without sizer.

panel/2

panel(Dir, Sub) -> any()

Create a panel (box) with given sizer orientation (either vertical or horizontal; wxBoxSizer will be used). This widget honors the sign of the orientation flag: negative values cause reverse order of placement (right to left for ?wxHORIZONTAL, and bottom to top for ?wxVERTICAL).

passEvent/2

passEvent(R, D) -> any()

Pass an event to the event link for this widget.

rcomp/2

rcomp(Sub, C) -> any()

Same as comp/2, but placement is reversed.

textLabel/2

textLabel(Fmt, T) -> any()

Create a text label that can be updated. Any event received will be formatted as specified by the first argument and displayed. The second argument specifies the initial text displayed.

topFrame/5

topFrame(Title, X, Y, Dir, Sub) -> any()

Create a toplevel frame (window) with given title, dimensions, and sizer orientation (either vertical or horizontal; wxBoxSizer will be used). This widget honors the sign of the orientation flag: negative values cause reverse order of placement (right to left for ?wxHORIZONTAL, and bottom to top for ?wxVERTICAL).
